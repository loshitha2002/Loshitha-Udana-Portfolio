## 🔧 **Customization Guide**

### **Update Personal Information**
1. **Profile Photo**: Replace `assets/profile.jpg` with your photo
2. **Contact Details**: Update phone, email, and social links in `index.html`
3. **Projects**: Modify project descriptions and GitHub links
4. **Achievements**: Update with your accomplishments

### **Color Scheme Customization**
```css
/* Primary colors in styles.css */
--primary-dark: #1a1a2e;
--secondary-dark: #16213e;
--accent-dark: #0f3460;
--accent-pink: #ff6b9d;
--accent-teal: #00d2d3;
```

### **Animation Preferences**
```css
/* Disable animations for accessibility */
@media (prefers-reduced-motion: reduce) {
  * { animation: none !important; }
}
```
